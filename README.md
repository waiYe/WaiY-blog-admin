基于vue-admin-template的个人博客管理后台，用于对前台个人博客进行管理，具有登录、博客信息、博客数量展示等功能。

### 前端主要使用的库/框架
- Vue-cli3
- element-ui
- vue-router
- Vuex
- axios

### 后端主要使用的库/框架
- node.js
- express
- mongoose

### 数据库
- mongodb

## 安装
`npm install`

## 启动
`npm run dev`
